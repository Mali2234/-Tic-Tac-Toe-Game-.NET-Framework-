# Tic-Tac-Toe (.NET Framework)

A classic Tic-Tac-Toe game built using C# and the .NET Framework (Windows Forms). This project demonstrates event-driven programming, UI design, and simple game-state management with clear, readable code. Players take turns placing X’s and O’s on a 3×3 grid while the game checks for wins, draws, and allows easy resetting.

---

## Game Features
- Player vs. Computer (basic AI)
- Player vs. Player gameplay
- Easy-to-read and organized code
- Draw (tie) detection
- Win detection (rows, columns, diagonals)
- Simple but clean AI logic using random move generation
- Resizable UI using TableLayoutPanel

---

## Technologies Used
- C#
- Windows Forms (.NET Framework)
- Visual Studio
